Famous Quotes API Trivia

This was our first group project after finishing the Javascript section of bootcamp. It's a game application utlizing two different quotes APIs to randomly generate quotes and possible answers for the user to guess. Utilizing OOP, it keeps track of usernames and scores adds the user to the leaderboard if they have a high score. It was built with Webpack and we were able to add some JS libraries for animations and sound effects.
